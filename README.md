# Laptop-Home-Server 

Design and Implementation of an On-Premise Home Linux Server in CasaOS Platform: Centralized Network Management, Security, and Cloud Independence

> [!NOTE]
> Το `irm` χρησιμοποιείται για τη λήψη του script, ενώ το `iex` για την εκτέλεσή του. Πάντα να ελέγχετε τα URLs.

> [!TIP]
> Μερικοί πάροχοι ISP μπλοκάρουν την πρόσβαση. Μπορείτε να το παρακάμψετε ενεργοποιώντας το DNS-over-HTTPS (DoH).

> [!WARNING]
> Προσοχή σε τρίτους που διανέμουν κακόβουλο λογισμικό!
